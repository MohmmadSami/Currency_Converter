### Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK (>=3.0.0)

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/currency-converter.git
```

2. Navigate to project directory
```bash
cd currency-converter
```

3. Get dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```

### Build APK

```bash

```

APK location: `build/app/outputs/flutter-apk/app-release.apk`
